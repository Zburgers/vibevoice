# VibeVoice 0.2.4 Release Notes

Release date: 2026-07-05

VibeVoice 0.2.4 hardens the production command surface, improves repeated-recording cleanup, refreshes the README with current screenshots, and keeps the in-app auto-update experience available for all users.

## Security

- Disabled in-app setup script execution in production builds.
- Restricted development setup execution to trusted app or resource paths.
- Tightened release-page URL validation to reject prefix-spoofed URLs.
- Added regression tests for setup gating and release URL validation.

## Improved

- Removed temporary recording and transcript artifacts after processing.
- Production UI now directs users to release installers or documented setup commands instead of running setup scripts.
- Diagnostics still checks GitHub Releases and preserves signed updater installation when release metadata is available.

## Documentation

- Rebuilt the README with current app screenshots, quick start commands, privacy notes, and update behavior.
- Updated install and architecture docs to match production setup hardening.

## Release

- App metadata updated to version `0.2.4`.
