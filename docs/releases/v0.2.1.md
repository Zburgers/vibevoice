# VibeVoice 0.2.1 Release Notes

Release date: 2026-07-02

## Summary

VibeVoice 0.2.1 is a simplification and polish release. The app now presents a focused Control, Settings, Library, and Diagnostics surface, removes the self-update subsystem, and keeps recording startup responsive by preparing audio capture on a worker.

## What Changed

- Added a `Preparing` state so start-recording returns immediately while engine and audio setup run off the UI command path.
- Added `vibevoice-meter-changed` events for mic-level updates; the frontend no longer polls `get_app_state` every second while recording.
- Updated automatic transcript completion states so successful output can show `Inserted` or `Copied` instead of jumping straight back to `Ready`.
- Removed source-control update commands, the Updates tab, bundled update scripts, and updater resources.
- Redesigned the main window into four simplified views with a responsive icon rail and browser-preview-safe rendering for visual checks.
- Rebuilt the floating pill controls with clear move, collapse, record, paste, and open-app actions.
- Made saved transcript history opt-in by default and added a Library clear-history action.
- Added setup command visibility and detected input-device details in diagnostics.
- Expanded Linux installer and readiness checks to include paste helpers such as `wtype`, `xdotool`, `xsel`, and `ydotool` status.
- Upgraded the Vite toolchain to clear the esbuild dev-server advisory and keep build tools in `devDependencies`.
- Updated app metadata to version `0.2.1`.

## User Impact

- Pressing record should no longer freeze while the app resolves the engine or starts audio capture.
- The app surface is smaller and more direct: primary recording controls are first, and history/dictionary live together in Library.
- History remains local, but it is no longer saved unless explicitly enabled.
- The setup panel shows the command it runs, making install behavior easier to audit.
- Developers update through git or release installers instead of an in-app updater.
