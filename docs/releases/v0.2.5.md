# VibeVoice 0.2.5

VibeVoice 0.2.5 focuses on local history controls, cheaper state refreshes, and release workflow reliability.

## Fixes

- Fixed the GitHub Actions release workflow version-read step so release jobs can reach the Tauri asset publishing step. Resolves #13.
- Cached diagnostics briefly and added explicit diagnostics refresh invalidation for engine path changes. Resolves #6.
- Debounced path-setting saves and only re-register the global hotkey when the hotkey value changes. Resolves #7.

## History

- Added local saved-history exports for Markdown and JSON from the Library view. Resolves #5.
- Added retention controls for maximum saved transcripts and optional age-based cleanup. Resolves #9.

## Verification

- `npm --prefix app run build`
- `cd app/src-tauri && cargo test --no-default-features`
