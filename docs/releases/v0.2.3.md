# VibeVoice 0.2.3 Release Notes

Release date: 2026-07-03

VibeVoice 0.2.3 fixes desktop-shell scrolling and collapsed navigation polish, then adds a Diagnostics update surface for checking and installing newer releases.

## Fixed

- Kept the custom titlebar fixed while the app content scrolls.
- Prevented the collapsed left rail controls from leaking outside the sidebar.
- Stopped document-level scrolling so only the main content pane owns overflow.

## Added

- Diagnostics now checks GitHub Releases for the latest public version.
- Diagnostics shows installed and latest versions, with a release-page fallback.
- Added Tauri signed-updater wiring for in-app installs when release updater metadata is available.

## Release

- Release workflow now publishes updater JSON and signatures for signed installer updates.
- App metadata updated to version `0.2.3`.
