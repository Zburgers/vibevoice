# VibeVoice 0.3 Compatibility Matrix

**Status:** Planning baseline; execution not started
**Owner:** Release lead
**Specification:** [Quality and Compatibility Contract](../specs/v0.3/quality-compatibility.md)

## Rules

A target is “supported” only after its exact platform/application combination completes:

- 50 single-line insertion trials
- 50 multiline insertion trials
- At least 99 successful insertions out of 100
- 50 forced target-change trials with zero automatic pastes
- 50 unchanged-clipboard restoration trials
- 50 newer-user-copy trials with zero overwritten newer values

Unsupported and blocked combinations remain visible. They are not removed from aggregate reporting.

Terminal Safe Mode is a P2/`0.3.x` gate. P0 still verifies that generic insertion never simulates Enter.

## Test record fields

Each completed row links a local or CI report containing:

```text
Date
Commit
Application version
OS and version
Display server
Display scale
Single or multiple monitors
VibeVoice pack and revision
Engine and model
Single-line passed / attempted
Multiline passed / attempted
Target-change automatic pastes / attempted
Clipboard restoration passed / attempted
Newer clipboard values overwritten / attempted
Known limitations
Evidence path or workflow URL
```
